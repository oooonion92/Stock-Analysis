# StockCodex 操盘方法论

更新：2026-06-03

本文用于以后每日复盘、持仓诊断和次日预案。核心目标不是预测涨跌，而是把交易先分清类型，再用结构、情绪、仓位三道过滤器决定是否执行。

## 1. 总纲

我们的系统分为两套打法：

1. 短线/超短：做主线、情绪、辨识度和资金聚焦，重视隔日预案、竞价反馈、分时承接和仓位速度。
2. 趋势/波段：做大方向、核心标的、级别趋势和仓位滚动，重视中枢结构、均线承接、破位规则和利润垫。

缠论只负责回答“走势结构到哪里了、买卖点是否成立、风险对应哪个级别”。它不单独给买入理由。短线买入必须有主线/情绪配合，趋势买入必须有大级别方向配合。

## 2. 短线打法

短线优先级：

1. 市场总情绪：指数是否放量、短线高度是否打开、连板晋级率和核按钮反馈如何。
2. 主线确认：是否有持续催化、完整梯队、前排强度、中军承接和后排扩散。
3. 核心标的：优先龙头、阵眼、情绪标、中军容量票，不碰无辨识度杂毛。
4. 买点位置：弱转强、分歧低吸、断板反包、均线/日内 VWAP 承接、前排不给机会时做中军。
5. 仓位纪律：主线明确才上仓位；轮动、电风扇、消息反抽、小作文题材只小仓试错。

短线常用判断：

- 情绪越冰，资金越容易集中到少数核心；但没有先手时不能追一致。
- 高标不是越高越能买，关键看换手、承接、监管压力、板块梯队和是否还有新主线接力。
- 普涨日不要乱追，优先找主线核心回踩或分歧日低吸。
- 新题材初期先看前排和梯队，前排买不到再看容量中军，后排只做套利。
- 没有完整梯队、没有带动性、没有持续催化的题材，按反抽或轮动处理。

短线卖点：

- 预期兑现、开盘不及预期、冲高无承接、核心标走弱、板块跟风断层，优先减仓。
- 高位缩量加速且无先手，更多是卖点或观察点，不是新增买点。
- 竞价超预期可作为买点，也可作为持仓冲高兑现点，取决于前一天是否已充分分歧。
- 盘中跌破日内均线/VWAP 后无法收回，短线执行降级。

## 3. 趋势波段打法

趋势优先级：

1. 大方向：是否属于当下机构/大资金愿意反复做的方向。
2. 核心标的：优先趋势龙头、中军、行业核心、容量标，不在边缘票上重仓。
3. 结构位置：是否处于大级别上涨、盘整上沿突破、回踩中枢/均线承接、二买/三买附近。
4. 仓位滚动：底仓跟随趋势，活动仓位做 T 或等结构买点补回。
5. 破位处理：趋势不破不看空；破位后先降低仓位，等重新站回趋势再加。

趋势仓位框架：

- 主升趋势：70%-80% 以上，允许盘中滚动。
- 正常调整：50%-60%，保留底仓和机动资金。
- 风险窗口：20%-30%，只保留核心底仓或观察仓。
- 下跌趋势：不做，除非出现明确大级别买点且仓位很小。

趋势操作纪律：

- 上涨时仓位大一点，下跌时仓位小一点。
- 没有突破重回上涨趋势前，不把已经减出去的仓位冲动加回去。
- 趋势票不按超短心态要求立刻涨；超短票也不能用趋势理由硬扛。
- 趋势核心可以用低级别卖点减仓，再用低级别买点回补。

## 4. 缠论走势判断

复盘固定使用两个级别：

- 30m：判断趋势/波段的主结构、方向和风险级别。
- 5m：判断短线执行、日内承接、二买/三买和止损位置。

缠论核心规则：

- 所有走势先分级别，再分上涨、下跌、盘整。
- 一个完成走势至少包含一个中枢；一个趋势至少包含两个同向中枢。
- 没有趋势，没有背驰。盘整里只能说盘整背驰或震荡衰竭，不能当趋势背驰乱用。
- 第一类买点来自下跌趋势背驰；第二类买点是第一类买点后的第一次次级别回调；第三类买点是离开中枢后回踩不回中枢。
- 大级别第二类买点，通常要到次级别找第一类买点精确定位。
- 大级别买点介入后，可以用次级别第一类卖点减仓，再用次级别第一类买点回补。

走势分解模板：

```text
先定级别：当前讨论的是 5m、30m、日线还是周线？
再定类型：上涨 / 下跌 / 盘整？
再找中枢：当前价格在中枢上方、内部、下方，还是回踩中枢？
再看动能：MACD 面积、DIF 斜率、红绿柱是否背驰或衰竭？
最后定动作：一买 / 二买 / 三买 / 一卖 / 二卖 / 观察 / 防守
```

## 5. 复盘流程

每日复盘按以下顺序执行：

1. 大盘总闸：指数位置、量能、5m/30m 中枢、DIF 方向、系统风险颜色。
2. 市场生态：短线高度、连板反馈、主线强度、机构趋势方向、是否轮动。
3. 持仓分流：每个持仓先标记为短线、趋势、打野、防守，不混用卖出逻辑。
4. 个股结构：5m 中枢位置、30m 趋势方向、背驰条件、VWAP/均线承接。
5. 题材地位：主线、支线、补涨、套利、纯消息反抽，分别给不同仓位上限。
6. 次日预案：红/黄/蓝三条路径，每条路径对应买、卖、减仓、观察条件。
7. 复盘纠错：记录今天是否追高、错过预案、仓位失控、把短线当趋势硬扛。

## 6. 执行过滤器

任何新增买入必须同时回答：

1. 它是什么打法：短线还是趋势？
2. 它处在哪个级别：5m 执行，30m 趋势，还是日线波段？
3. 它的市场理由是什么：主线、情绪、机构趋势、补涨，还是套利？
4. 它的结构买点是什么：回踩、二买、三买、弱转强、断板反包？
5. 错了在哪里卖：中枢下沿、VWAP、均线、前低、竞价不及预期？
6. 仓位为什么是这个比例：主线确定性、利润垫、风险窗口、是否有先手？

如果无法回答其中任意三项，默认不买；如果只是想参与热闹，只允许小仓试错。

## 7. 以后复盘中的固定输出

以后 StockCodex 复盘要固定增加两句结论：

```text
交易类型：短线 / 趋势 / 打野 / 防守
执行结论：上仓位 / 持有做T / 等回踩 / 减仓 / 清仓 / 只观察
```

对短线标的，优先输出：

```text
主线地位、情绪锚、前排/中军/后排、竞价预期、分时承接、隔日卖点
```

对趋势标的，优先输出：

```text
30m方向、5m买卖点、中枢上下沿、均线/VWAP承接、底仓与活动仓比例、破位条件
```

最终原则：先活下来，再稳定赚；先控制回撤，再提高收益；先等买点，再谈观点。
