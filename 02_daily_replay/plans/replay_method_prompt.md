# StockCodex 复盘调用准则

每次复盘先执行这套分流：

```text
1. 先判大盘总闸：红 / 黄 / 蓝。
2. 再判市场生态：主线趋势 / 短线情绪 / 轮动套利 / 风险窗口。
3. 再给每个标的贴标签：短线 / 趋势 / 打野 / 防守。
4. 短线看主线、情绪锚、辨识度、竞价、VWAP/均线承接、隔日卖点。
5. 趋势看30m方向、5m买卖点、中枢、均线承接、底仓和活动仓。
6. 缠论只确认结构，不单独构成买入理由。
7. 输出必须给：交易类型、仓位建议、买点条件、卖点条件、失效条件。
```

固定纪律：

```text
09:30-10:00 不追买。
大盘总闸蓝色时，所有新增买点降级为观察。
没有主线地位的短线票不重仓。
趋势票不按超短心态乱卖，超短票不按趋势理由硬扛。
没有突破重回趋势前，不把已减仓位冲动加回。
只在可解释、可止损、可复盘的位置动手。
```
