# StockCodex 复盘调用准则

每次复盘先执行这套分流：

```text
1. 先判大盘总闸：红 / 黄 / 蓝。
2. 先锁定最新结构：最近有效离开段 / 最近活跃中枢 / 最近底分型，旧中枢只做失败确认，不抢最新结构的话语权。
3. 再判市场生态：主线趋势 / 短线情绪 / 轮动套利 / 风险窗口。
4. 再给每个标的贴标签：短线 / 趋势 / 打野 / 防守。
5. 短线看主线、情绪锚、辨识度、竞价、VWAP/均线承接、隔日卖点。
6. 趋势看30m方向、5m买卖点、中枢、均线承接、底仓和活动仓。
7. 缠论只确认结构，不单独构成买入理由。
8. 输出必须给：交易类型、仓位建议、买点条件、卖点条件、失效条件。
```

最新结构优先规则：

```text
1. 如果价格已经离开5m中枢，先讨论离开段是否延续、回踩承接是否成立，不再先说“站上老中枢上沿才转强”。
2. 旧5m中枢上沿/下沿只作为最后降级线、失败确认线，不能替代当前离开段或最新底分型。
3. 红黄蓝路径必须锚定最新有效结构：离开段、最新底分型、最新承接区，不用上一段结构偷换当前观察位。
4. 若30m仍在修复但5m已向上离开，中短结论应写“延续待确认”或“回踩验证”，而不是退回纯中枢思维。
5. 若价格重新跌回老中枢内，才允许把老中枢重新升格为主观察结构。
```

固定纪律：

```text
09:30-10:00 不追买。
大盘总闸蓝色时，所有新增买点降级为观察。
没有主线地位的短线票不重仓。
趋势票不按超短心态乱卖，超短票不按趋势理由硬扛。
没有突破重回趋势前，不把已减仓位冲动加回。
只在可解释、可止损、可复盘的位置动手。
```
